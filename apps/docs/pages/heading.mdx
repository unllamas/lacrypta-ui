# Heading
Heading is used to render semantic HTML heading elements.

## Import

```js
import { Heading } from '@unllamas/lacrypta-ui'
```

## Usage

Heading composes Box so you can use all the style props and add responsive styles as well. It renders an `<h2>` tag by default.

<br />

<Card variant='outline' spacing={4}>
  <Heading>I'm a Heading</Heading>
</Card>

```js
<Heading>I'm a Heading</Heading>
```

### Changing visual size

<Callout type="error">Falta implementar</Callout>

To increase the visual size of the heading, use the `size` props. This property ensures that the heading size automatically adjusts for smaller screens.

<Callout type="warning">Corregir `line-hegiht`</Callout>

<br />

<Card variant='outline' spacing={4}>
  <Heading as='h1' size='4xl'>(4xl) In love with React & Next</Heading>
  <Heading as='h2' size='3xl'>(3xl) In love with React & Next</Heading>
  <Heading as='h3' size='2xl'>(2xl) In love with React & Next</Heading>

  <Heading as='h5' size='md'>(md) In love with React & Next</Heading>

</Card>
