# Heading

Heading is used to render semantic HTML heading elements.

## Import

```js copy
import { Heading } from '@unllamas/lacrypta-ui';
```

## Usage

Heading composes Box so you can use all the style props and add responsive styles as well. It renders an `<h1>` tag by default.

```js copy
<Heading>I'm a Heading</Heading>
```

### Heading As

The `as` property receives `h1`, `h2`, `...` to `h6`.

```js copy
<Heading as='h1'>(h1) In love with React & Next</Heading>
<Heading as='h2'>(h2) In love with React & Next</Heading>
<Heading as='h3'>(h3) In love with React & Next</Heading>
<Heading as='h4'>(h4) In love with React & Next</Heading>
<Heading as='h5'>(h5) In love with React & Next</Heading>
<Heading as='h6'>(h6) In love with React & Next</Heading>
```
