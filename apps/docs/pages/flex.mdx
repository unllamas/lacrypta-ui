# Flex

Flex is a layout component used to group elements together and apply a space between them.

## Import

```js copy
import { Flex } from '@unllamas/lacrypta-ui';
```

## Usage

<br />

<Card variant="outline" spacing={4}>
  <Flex>
    <Button>Button</Button>
    <Button>Button</Button>
  </Flex>
</Card>

```js copy
<Flex>
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```

### Flex Gap

The `gap` property receives `xs`, `sm` or `md`

<Callout type="warning">
  Los valores `0`, `2`, `4`, `8` y `16` ser√°n deprecados.
</Callout>

<br />

<Card variant="outline" spacing={4}>
  <Flex gap="md">
    <Button>Button</Button>
    <Button>Button</Button>
  </Flex>
</Card>

```js copy /gap/
<Flex gap="md">
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```

### Flex Justify

The `justify` property receives `start`, `end`, `center` or `space-between`

<br />

<Card variant="outline" spacing={4}>
  <Flex gap="md" justify="center">
    <Button>Button</Button>
    <Button>Button</Button>
  </Flex>
</Card>

```js copy /justify/
<Flex ... justify="center">
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```

### Flex Direction

The `direction` property receives `row` or `column`

<br />

<Card variant="outline" spacing={4}>
  <Flex gap="md" direction="column">
    <Button>Button</Button>
    <Button>Button</Button>
  </Flex>
</Card>

```js copy /direction/
<Flex ... direction="column">
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```

### Flex Align

The `align` property receives `start`, `end` or `center`

<br />

<Card variant="outline" spacing={4}>
  <Flex gap="md" direction="column" align="center">
    <Button>Button</Button>
    <Button>Button</Button>
  </Flex>
</Card>

```js copy /align/
<Flex ... align="center">
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```
