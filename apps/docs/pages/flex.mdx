# Flex

Flex is a layout component used to group elements together and apply a space between them.

## Import 

```js
import { Flex } from '@unllamas/lacrypta-ui'
```

## Usage

<br />

<Card variant='outline' spacing={4}>
  <Container size='sm'>
    <Flex>
      <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua</Text>
      <Button>Button</Button>
    </Flex>
  </Container>
</Card>

```js
<Flex>
  <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua</Text>
  <Button>Button</Button>
</Flex>
```

### Flex Direction

The direction property receives `row` or `column` as a value

<br />

<Card variant='outline' spacing={4}>
  <Container size='sm'>
    <Flex gap='md' direction='column' >
      <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua</Text>
      <Button>Button</Button>
    </Flex>
  </Container>
</Card>

```js
<Flex gap='md' direction='column'>

  <Button>Button</Button>
```

### Flex Align

The align property receives `start`, `end` or `center`

<br />

<Card variant='outline' spacing={4}>
  <Container size='sm'>
    <Flex gap='md' direction='column' align='center'>
      <Text align='center'>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et.</Text>
      <Button>Button</Button>
    </Flex>
  </Container>
</Card>

```js
<Flex gap='md' direction='column' align='center'>
  <Text align='center'>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et.</Text>
  <Button>Button</Button>
</Flex>
```

### Flex Justify

The justify property receives `start`, `end`, `center` or `space-between`

<br />

<Card variant='outline' spacing={4}>
  <Container size='sm'>
    <Flex gap='md' justify='center'>
      <Button>Button</Button>
      <Button>Button</Button>
    </Flex>
  </Container>
</Card>

```js
<Flex gap='md' justify='center'>
  <Button>Button</Button>
  <Button>Button</Button>
</Flex>
```
