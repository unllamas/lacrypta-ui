# Text

Text is the used to render text and paragraphs within an interface.

## Import

```js
import { Text } from '@unllamas/lacrypta-ui'
```

## Usage

<br />

<Card variant='outline' spacing={4}>
  <Text>In love with React & Next</Text>
</Card>

```js
<Text>In love with React & Next</Text>
```

### Text Size

To increase the visual size of the heading, use the `size` props. This property ensures that the heading size automatically adjusts for smaller screens.

<Callout type="warning">Deprecados los valores `small` y `normal`</Callout>

<br />

<Card variant='outline' spacing={4}>
  <Flex direction='column' gap={16}>
    <Text size='md'>(md) In love with React & Next</Text>
    <Text size='sm'>(sm) In love with React & Next</Text>
    <Text size='xs'>(xs) In love with React & Next</Text>
  </Flex>
</Card>

```js
<Text size='md'>(md) In love with React & Next</Text>
<Text size='sm'>(sm) In love with React & Next</Text>
<Text size='xs'>(xs) In love with React & Next</Text>
```

### Text Align

The align property receives `left`, `center` or `right` value

<br />

<Card variant='outline' spacing={4}>
  <Text align='left'>In love with React & Next</Text>
  <Text align='center'>In love with React & Next</Text>
  <Text align='right'>In love with React & Next</Text>
</Card>

```js
<Text align='left'>In love with React & Next</Text>
<Text align='center'>In love with React & Next</Text>
<Text align='right'>In love with React & Next</Text>
```

### Text Color

The color property receives a hexadecimal color

<br />

<Card variant='outline' spacing={4}>
  <Text color='yellow'>In love with React & Next</Text>
</Card>

```js
<Text color='yellow'>In love with React & Next</Text>
```

### Override the element

<Callout type="error">Falta implementar</Callout>

To override the element that gets rendered, pass the as prop. Use Inspect Element to see the element that gets rendered in html.

<br />

<Card variant='outline' spacing={4}>
  <Flex direction='column' gap={16}>
    <Text isBold>Bold</Text>
    <Text as='i'>Italic</Text>
    <Text as='u'>Underline</Text>
    <Text as='abbr'>I18N</Text>
    <Text as='cite'>Citation</Text>
    <Text as='del'>Deleted</Text>
    <Text as='em'>Emphasis</Text>
    <Text as='ins'>Inserted</Text>
    <Text as='kbd'>Ctrl + C</Text>
    <Text as='mark'>Highlighted</Text>
    <Text as='s'>Strikethrough</Text>
    <Text as='samp'>Sample</Text>
    <Text as='sub'>sub</Text>
    <Text as='sup'>sup</Text>
  </Flex>
</Card>

```js
<Text isBold>Bold</Text>
<Text as='i'>Italic</Text>
<Text as='u'>Underline</Text>
<Text as='abbr'>I18N</Text>
<Text as='cite'>Citation</Text>
<Text as='del'>Deleted</Text>
<Text as='em'>Emphasis</Text>
<Text as='ins'>Inserted</Text>
<Text as='kbd'>Ctrl + C</Text>
<Text as='mark'>Highlighted</Text>
<Text as='s'>Strikethrough</Text>
<Text as='samp'>Sample</Text>
<Text as='sub'>sub</Text>
<Text as='sup'>sup</Text>
```
