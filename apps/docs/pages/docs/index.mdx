# Introducción

Componentes bellamente diseñados que puedes copiar y pegar en tus aplicaciones. Accesibles. Personalizables. Código abierto.

### ¿Qué es `lacrypta-ui`?

`lacrypta-ui` es una biblioteca de interfaz de usuario para NextJS que te ayuda a construir interfaces de usuario hermosas y accesibles.

El objetivo principal de `lacrypta-ui` es agilizar el proceso de desarrollo, ofreciendo un diseño de sistema hermoso y adaptable para una experiencia de usuario mejorada.
